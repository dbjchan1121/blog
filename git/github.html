# 配置git proxy（需要有本地代理）

git config --global --unset https.proxy

# 解决 VS Code 仍然提示输入密码

设置 IdentityFile 的原因是确保能找到密钥位置

Host *
  UseKeychain yes
  AddKeysToAgent yes
  IdentityFile ~/.ssh/id_rsa
